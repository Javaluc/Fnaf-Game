# FNAF Game

## Project Description
This project is a fan-made game inspired by the popular horror franchise, Five Nights at Freddy's. Players take on the role of a night security guard, monitoring the movements of animatronic characters and ensuring their safety through various game mechanics.

## Setup Instructions
1. **Clone the Repository**:  
   Use the command `git clone https://github.com/Javaluc/Fnaf-Game.git` to clone the repository to your local machine.
2. **Install Dependencies**:  
   Navigate to the project directory and run `npm install` to install all required dependencies.
3. **Run the Game**:  
   Use the command `npm start` to launch the game in your browser.

## Gameplay Information
- Players must survive each night while managing resources like power and monitoring the actions of animatronics.
- The game includes various features such as camera monitoring, doors, and light controls to prevent animatronic attacks.
- Each night presents increasing difficulty and new challenges, keeping gameplay engaging and thrilling.

Happy gaming!
